# Markdown_table

convert markDown to html table


format of text should be like this:
| Syntax      | Description |
| Header      | Title       |
| Paragraph   | Text        |

it convert markdown to html table like this:

<table>
    <tbody>
        <tr>
            <td>Syntax</td>
            <td>Description</td>
        </tr>
        <tr>
            <td>Header</td>
            <td>Title</td>
        </tr>
        <tr>
            <td>Paragraph</td>
            <td>Text</td>
        </tr>
    </tbody>
</table>
